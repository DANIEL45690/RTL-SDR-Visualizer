## 📡 Что это?

RealSDRVisualizer — это десктопное приложение на Java для визуализации спектра радиосигналов в реальном времени с использованием RTL-SDR приёмников. Программа подключается к `rtl_tcp` серверу, получает I/Q сэмплы, выполняет БПФ (FFT) и отображает спектр мощности.

**Назначение:** Визуализация радиочастотного спектра, поиск сигналов, анализ уровней шума.

---

## ✨ Возможности

### Основные
- 📡 **Подключение к rtl_tcp** (стандартный протокол RTL-SDR)
- 📊 **Визуализация спектра** в реальном времени (обновление ~20 FPS)
- 🎛️ **Управление частотой** — установка центральной частоты
- 🔊 **Регулировка усиления** (ползунок от 0 до 100)
- 📈 **Определение пиковой частоты** автоматически
- 📉 **Расчёт уровня шума** и средней мощности

### Технические характеристики
| Параметр | Значение |
|----------|----------|
| Частота дискретизации | 2.4 MSPS |
| Размер FFT | 4096 точек |
| Разрешение по частоте | ~586 Гц |
| Битность сэмплов | 8 бит (I/Q) |
| Частота обновления | ~20 Гц |

### Визуализация
- 🌊 **Спектр мощности** (график амплитуды)
- 🔴 **Маркер пикового сигнала**
- 📊 **Сетка частот** для ориентации
- 🎨 **Терминальный стиль** оформления (ASCII-рамки)
