RTL-SDR ВИЗУАЛИЗАТОР СПЕКТРА

ПОЛНОЕ РУКОВОДСТВО ПО ЗАПУСКУ
ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ

Данное руководство предназначено для работы с физическим устройством RTL-SDR.
Без USB-приёмника на базе чипа RTL2832U программа НЕ БУДЕТ РАБОТАТЬ.
Убедитесь, что у вас есть совместимое устройство перед началом настройки.

ТРЕБОВАНИЯ

  • RTL-SDR USB донгл (RTL2832U)
  • Windows 10 / 11 (64-bit)
  • Java JDK 11 или новее
  • Свободный USB порт
  • Антенна (входит в комплект)

ВОЗМОЖНОСТИ

  • Спектр до 2.4 МГц в реальном времени
  • Диапазон частот: 24 - 1700 МГц
  • БПФ 4096 точек
  • Визуализация I/Q сигнала
  • Регулировка усиления

СТАТУС

→ Устройство: не подключено
→ Драйвер: требуется установка
→ rtl_tcp: ожидание запуска
→ Java: проверьте версию
→ Компиляция: javac RealSDRVisualizer.java
1. УСТАНОВКА ДРАЙВЕРОВ (ZADIG)
  1. Скачайте Zadig с официального сайта: https://zadig.akeo.ie/
  2. Подключите RTL-SDR к компьютеру
  3. Запустите Zadig от имени администратора
  4. В меню выберите Options → List All Devices
  5. Найдите в списке Bulk-In, Interface (Interface 0)
  6. В поле выбора драйвера укажите WinUSB
  7. Нажмите кнопку Install Driver
  8. Дождитесь завершения установки
После установки драйвера устройство должно определиться в системе.
2. УСТАНОВКА RTL-SDR УТИЛИТ

Способ 1: через Chocolatey (рекомендуется)

# Установка Chocolatey (от имени администратора)
Set-ExecutionPolicy Bypass -Scope Process -Force
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# Установка rtl-sdr
choco install rtl-sdr -y

Способ 2: ручная установка

1. Скачайте сборку: https://github.com/harryjyoung/rtl-sdr-windows/releases
2. Распакуйте архив в папку C:\rtl-sdr
3. Добавьте C:\rtl-sdr\bin в переменную PATH
   (Панель управления → Система → Дополнительные параметры → Переменные среды)

Проверка установки

# В командной строке выполните:
rtl_test -t

# Если устройство найдено, вы увидите информацию о нём
3. ПРОВЕРКА JAVA

Проверьте версию Java

java -version

Если Java не установлена:

1. Скачайте Java 17 LTS: https://adoptium.net/temurin/releases/
2. Выберите: Windows x64 .msi Installer
3. Установите и перезагрузите компьютер
4. JAVA ВИЗУАЛИЗАТОР

Создайте файл RealSDRVisualizer.java

Скопируйте код ниже в файл:

import javax.swing.*;
import java.awt.*;
import java.io.*;
import java.net.*;

public class RealSDRVisualizer extends JPanel implements Runnable {
    private static final int WIDTH = 1280;
    private static final int HEIGHT = 720;
    private Socket socket;
    private DataInputStream dis;
    private DataOutputStream dos;
    private volatile boolean running = true;
    private float[] iqBuffer = new float[131072];
    private int bufferPos = 0;
    private float[] real = new float[4096];
    private float[] imag = new float[4096];
    private int[] spectrum = new int[2048];
    private double centerFreq = 100000000;
    private JFrame frame;
    private JTextField freqField;
    private JLabel statusLabel;

    public RealSDRVisualizer() {
        setPreferredSize(new Dimension(WIDTH, HEIGHT));
        setBackground(Color.WHITE);
        createUI();
        connect();
        new Thread(this).start();
    }

    private void createUI() {
        frame = new JFrame("RTL-SDR Visualizer by @console_hack");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        JPanel top = new JPanel();
        top.setBackground(Color.WHITE);
        top.setBorder(BorderFactory.createLineBorder(Color.BLACK));

        freqField = new JTextField("100.0", 10);
        JButton setFreq = new JButton("Set Frequency");
        statusLabel = new JLabel("Status: Connecting...");

        setFreq.addActionListener(e -> setFrequency());

        top.add(new JLabel("Frequency (MHz):"));
        top.add(freqField);
        top.add(setFreq);
        top.add(statusLabel);

        frame.add(top, BorderLayout.NORTH);
        frame.add(this, BorderLayout.CENTER);
        frame.pack();
        frame.setLocationRelativeTo(null);
        frame.setVisible(true);
    }

    private void connect() {
        try {
            socket = new Socket("127.0.0.1", 1234);
            dis = new DataInputStream(socket.getInputStream());
            dos = new DataOutputStream(socket.getOutputStream());
            dos.writeByte(0x01);
            dos.writeInt(0);
            dos.writeInt(2400000);
            dos.writeInt(0);
            dos.flush();
            statusLabel.setText("Status: Connected");

            new Thread(() -> {
                byte[] raw = new byte[131072];
                while (running) {
                    try {
                        int read = dis.read(raw);
                        for (int i = 0; i < read/2 && bufferPos < iqBuffer.length; i++) {
                            iqBuffer[bufferPos++] = (raw[i*2] / 128.0f);
                            iqBuffer[bufferPos++] = (raw[i*2+1] / 128.0f);
                            if (bufferPos >= iqBuffer.length) bufferPos = 0;
                        }
                    } catch (IOException e) {
                        statusLabel.setText("Status: Connection lost");
                    }
                }
            }).start();
        } catch (IOException e) {
            statusLabel.setText("Status: rtl_tcp not running");
        }
    }

    private void setFrequency() {
        try {
            centerFreq = (long)(Double.parseDouble(freqField.getText()) * 1e6);
            dos.writeByte(0x02);
            dos.writeInt((int)centerFreq);
            dos.flush();
            statusLabel.setText(String.format("Status: Tuned to %.3f MHz", centerFreq/1e6));
        } catch (Exception e) {
            statusLabel.setText("Status: Frequency error");
        }
    }

    private void processFFT() {
        if (bufferPos < 8192) return;

        for (int i = 0; i < 4096; i++) {
            int idx = (bufferPos - 8192 + i*2) % iqBuffer.length;
            if (idx >= 0 && idx + 1 < iqBuffer.length) {
                real[i] = iqBuffer[idx];
                imag[i] = iqBuffer[idx + 1];
            }
        }

        fft(real, imag);

        for (int i = 0; i < 2048; i++) {
            double mag = Math.hypot(real[i], imag[i]);
            spectrum[i] = (int)Math.min(350, mag * 300);
        }
        repaint();
    }

    private void fft(float[] r, float[] im) {
        int n = 4096;
        int bits = 0;
        while ((1 << bits) < n) bits++;

        for (int i = 0; i < n; i++) {
            int rev = 0;
            for (int j = 0; j < bits; j++) {
                rev = (rev << 1) | ((i >> j) & 1);
            }
            if (i < rev) {
                float tr = r[i]; r[i] = r[rev]; r[rev] = tr;
                float ti = im[i]; im[i] = im[rev]; im[rev] = ti;
            }
        }

        for (int len = 2; len <= n; len <<= 1) {
            float angle = (float)(-2 * Math.PI / len);
            float wrlen = (float)Math.cos(angle);
            float wilen = (float)Math.sin(angle);

            for (int i = 0; i < n; i += len) {
                float wr = 1;
                float wi = 0;
                for (int j = 0; j < len/2; j++) {
                    float ur = r[i+j];
                    float ui = im[i+j];
                    float vr = r[i+j+len/2] * wr - im[i+j+len/2] * wi;
                    float vi = r[i+j+len/2] * wi + im[i+j+len/2] * wr;
                    r[i+j] = ur + vr;
                    im[i+j] = ui + vi;
                    r[i+j+len/2] = ur - vr;
                    im[i+j+len/2] = ui - vi;
                    float nwr = wr * wrlen - wi * wilen;
                    float nwi = wr * wilen + wi * wrlen;
                    wr = nwr;
                    wi = nwi;
                }
            }
        }
    }

    public void run() {
        while (running) {
            if (dis != null) processFFT();
            try { Thread.sleep(33); } catch (Exception e) {}
        }
    }

    protected void paintComponent(Graphics g) {
        super.paintComponent(g);
        g.setColor(Color.WHITE);
        g.fillRect(0, 0, WIDTH, HEIGHT);

        g.setColor(Color.BLACK);
        for (int x = 0; x < WIDTH; x++) {
            int idx = (x * 2048) / WIDTH;
            if (idx < 2048) {
                int h = Math.min(HEIGHT - 100, spectrum[idx]);
                g.drawLine(x, HEIGHT - 60, x, HEIGHT - 60 - h);
            }
        }

        g.setFont(new Font("Intel One Mono", Font.PLAIN, 12));
        g.drawString("RTL-SDR Spectrum Analyzer by @console_hack", 20, 30);
        g.drawString(String.format("Center: %.3f MHz | Span: 2.4 MHz", centerFreq/1e6), 20, 55);

        for (int i = 0; i <= 6; i++) {
            int x = 20 + (i * (WIDTH - 40) / 6);
            double freq = centerFreq - 1200000 + (i * 2400000 / 6);
            g.drawString(String.format("%.1f", freq/1e6), x - 15, HEIGHT - 25);
        }

        g.drawRect(10, 10, WIDTH - 20, HEIGHT - 20);
    }

    public static void main(String[] args) {
        new RealSDRVisualizer();
    }
}

Компиляция и запуск

javac RealSDRVisualizer.java
java RealSDRVisualizer
5. ЗАПУСК

Запустите rtl_tcp (первое окно терминала)

rtl_tcp -a 127.0.0.1 -p 1234 -f 100000000 -s 2400000 -g 40

Запустите визуализатор (второе окно терминала)

java RealSDRVisualizer

Использование

  • Введите частоту в МГц (например 100.5 для FM)
  • Нажмите "Set Frequency" для перестройки
  • Наблюдайте спектр сигналов в реальном времени
ЧАСТОТЫ ДЛЯ ТЕСТИРОВАНИЯ

FM-радио: 88-108 МГц
Авиадиапазон: 118-137 МГц
ДМВ (UHF): 400-470 МГц
GSM: 900 МГц, 1800 МГц
LTE: 2600 МГц
УСТРАНЕНИЕ НЕИСПРАВНОСТЕЙ

Ошибка: rtl_tcp не является командой
→ Установите rtl-sdr или добавьте папку bin в PATH

Ошибка: No RTL-SDR devices found
→ Переустановите драйвер через Zadig, перезагрузите компьютер

Ошибка: Connection refused
→ Сначала запустите rtl_tcp, только потом Java программу

Ошибка: cannot find symbol
→ Проверьте правильность копирования Java кода
ПРИЗНАКИ КОРРЕКТНОЙ РАБОТЫ

→ rtl_test показывает найденные устройства
→ rtl_tcp пишет "listening on 127.0.0.1:1234"
→ Java приложение показывает "Status: Connected"
→ В окне визуализатора отображается спектр сигналов
→ На FM-частотах наблюдаются характерные пики
